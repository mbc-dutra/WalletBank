# 💳 WalletBank System (Java)

Projeto simples de sistema bancário em Java, desenvolvido com foco em prática de lógica de programação, controle de fluxo e interação com o usuário via console.

---

## 🚀 Funcionalidades

O sistema permite:

* 🔍 Consultar saldo
* 💰 Receber valores (depósito)
* 📤 Transferir valores
* 🚪 Sair do sistema

---

## 🧠 Conceitos aplicados

Durante o desenvolvimento foram praticados:

* Estrutura de repetição (`while`)
* Estrutura de decisão (`switch-case`)
* Entrada de dados com `Scanner`
* Manipulação de variáveis (`double`, `int`)
* Validação de regras (ex: saldo insuficiente)
* Formatação de saída (`printf`)

---
